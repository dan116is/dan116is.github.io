# הבית שלי

אפליקציית ווב פרטית לניהול אישי ומשפחתי. רצה בדפדפן של הטלפון (PWA), שומרת
את הנתונים מקומית במכשיר בלבד, ועובדת גם לא־מקוון אחרי טעינה ראשונה.

## מה יש בפנים

- **תרופות** — מעקב מלאי, תאריכי תפוגה, מנות יומיות, התראות מראש.
- **קניות** — רשימה משותפת לפי קטגוריות, סימון מה נקנה, ניקוי בלחיצה.
- **משימות** — לוח שנה משפחתי, עדיפויות, תזכורות ליום עצמו ולמשימות באיחור.
- **תקציב** — הוצאות חודשיות לפי קטגוריה, יעד תקציבי, חישוב מה נשאר.
- **בני משפחה** — שיוך תרופות/משימות לבני המשפחה (כברירת מחדל: הילה, גפן, אורי).
- **גיבוי ושחזור** — ייצוא/ייבוא של כל הנתונים כקובץ JSON.

הכל בעברית, RTL, ערכת נושא כהה/בהירה לפי המכשיר.

## איך מריצים

### באופן מקומי

```bash
python3 -m http.server 8000
# פתח http://localhost:8000 בדפדפן
```

### פריסה חינמית ב־GitHub Pages

1. ב־`Settings` של הריפו → `Pages`
2. תחת `Source` בחר `Deploy from a branch` → ענף `main` → `/ (root)`
3. אחרי כמה דקות תקבל כתובת `https://<user>.github.io/<repo>/`
4. פתח את הכתובת בטלפון → תפריט הדפדפן → **הוסף למסך הבית**
5. האפליקציה תופיע כמו אפליקציה רגילה, עם שורת סטטוס מותאמת, ותעבוד גם בלי אינטרנט

## פרטיות

כל הנתונים נשמרים ב־`localStorage` של הדפדפן במכשיר שלך בלבד. שום דבר לא נשלח
לשרת. אם תנקה נתוני דפדפן או תחליף מכשיר — הנתונים ימחקו. השתמש בגיבוי
ידני (בהגדרות) כדי לשמור עותק חיצוני.
